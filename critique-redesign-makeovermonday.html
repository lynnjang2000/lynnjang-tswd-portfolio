Reason for Redesign

The reason I chose this map for redesign is that the level of visual attention varies greatly depending on each country’s geographic size. For example, Oceania has the second-highest average number of public holidays, but this is difficult to notice quickly because countries like Fiji appear so small on the map. In addition, the original map does not display the global average, which makes meaningful comparison more challenging.

To address these limitations, I aimed to create a visualization that provides a clearer and more actionable understanding of how different regions compare. I calculated and displayed both the global average and the average for each continent so viewers can immediately grasp structural differences that may influence scheduling, productivity, or cross-regional planning—insights that are not easily visible in a country-by-country map.

Process

My goal was to compare the average number of public holidays across continents and against the global average. First, I categorized each country by continent and added labels. Then, I calculated the average number of public holidays for each continent as well as the global average. Finally, I used a line and dot plot to display these values and highlighted the global average on the y-axis.
